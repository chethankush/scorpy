### what is the website about
```bash
This website is simple php program which
1) accepts username and password if not present it will ask for registration
2) Once registered it will login and ask you enter - lastname, firstname, Address and city and stores in the database
3) If you click on show data, it will take you to next page where it has all the data entered
4) There is option to delete the entry as well
5) As long the id is valid you will be logged in and you can click logout button to come out the website.
```
## Requirements
```bash
1) I have installed MySQL, php, apache2 and phpmyadmin on ubuntu Linux
2) I have created 2 DB where login data is maintained as "users" and another db as "register" where it holds person's data
```
